# HTB-Writeups
Мои отчёты по penetration testing на Hack The Box. Навыки: Nmap, Burp Suite, etc.
Отчеты и решения по машинам HTB

Это репозиторий, в котором я собираю свои отчеты, заметки и скрипты, связанные с прохождением машин на платформе Hack The Box (HTB). 

Цель проекта — демонстрация навыков а также практика в области информационной безопасности, анализа уязвимостей, использование системы контроля версий (Git).

## Что я делаю в этом проекте

- Подробные отчеты по каждой машине HTB, включающие описание уязвимостей, этапы взлома и использованные инструменты.
- Храню скрипты и полезные команды для автоматизации и повторного использования.
- Документирую свои шаги и решения

## Как использовать этот репозиторий

- Перейти в папки с названиями машин, чтобы найти соответствующие отчеты.
- Ознакомиться с файлами Markdown, PDF или скриптами, чтобы понять мои методы.
- В будущем планирую добавлять новые машины, скрипты и заметки.

## Мои навыки, продемонстрированные в этом проекте

- Работа с системами контроля версий (Git)
- Документирование и описание решений
- Анализ уязвимостей и эксплуатации
- Автоматизация задач с помощью скриптов
- Построение профессиональных отчетов и презентаций
